Website Sale No Cart Redirection

Este módulo modifica el comportamiento del sitio web al hacer click en el botón de "Añadir al carrito", manteniendo al usuario en la página actual y mostrando un mensaje informativo de producto añadido al carrito.

Uso

En el listado de productos, ahora, al añadir al carrito no redirige al carrito si no que se mantiene en la página actual y se muestra un aviso de producto añadido.
