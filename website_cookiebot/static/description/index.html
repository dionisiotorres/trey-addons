Cookiebot and GDPR integration

Este módulo integra la funcionalidad del Cookiebot y la RGPD con el sitio web de Odoo.

  • Créditos
    • Autor

Uso

  1. Ir a Sitio web > Configuración > Ajustes > Sitio web.
  2. Activar la casilla de verificación Cookiebot.
  3. Añadir el código Cookiebot.
  4. Hacer clic en el botón "Guardar".
