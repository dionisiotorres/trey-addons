Website Public Users

Allow to select between public users in website

Este módulo permite seleccionar entre diferentes usuarios públicos configurados en el sitio web, para adaptar idiomas, tarifas sin necesidad de estar logueado con una cuenta de usuario.

Uso

  • Cree los usuarios necesarios para el sitio web marcando la opción "Publico"
  • En la configuración del sitio web, seleccione los usuarios públicos disponibles
  • En el sitio web, estarán disponibles los distintos perfiles bajo un desplegable
