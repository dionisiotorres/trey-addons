Installation

This module depends on:
* product_brand
This module is part of the OCA/product_attribute branch.

Usage

  1. Go to Website > Settings > Website.
  2. In "Facebook Pixel" group.
  3. Fill the field "Company Brand" if your products don't have a specific brand.
  4. Click on "Save" button.
